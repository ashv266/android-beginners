Taught by Audrey Troutt / @auditty

Is this class for you?

You should be..

  • curious and excited to learn about how apps are built.
  • prepared to face challenges and have some fun along the way.
  • already have some experience with programming concepts like variables, methods, classes, arrays, conditionals, and loops.
You are not expected to have any previous Android or even Java experience for this class. However, to move things along it would help if this wasn't your very first introduction to programming in general. I think it is just as important to build things from scratch as it is to understand existing code and take things apart. Therefore the coding exercises will be a combination of read and edit, build from pieces, and start from scratch.

Get Code (Download)

You have two options for getting the code for this course: download or fork.

Downloading is simpler:

  1. Go to GitHub atroutt/AndroidBeginnerFirstApp
  2. On the top right press "Clone or download" and click "Download ZIP"
  3. Follow the same steps for GitHub atroutt/AndroidBeginnerChat
  4. On your computer, open the zip files with your favorite unzip program and put the 'AndroidBeginnerFirstApp' and 'AndroidBeginnerChat' folders somewhere you can find them later.

Get Code (Fork)

The alternate option for getting the course code is to fork it from GitHub.

Forking is more complex, but it has multiple benefits:

  • You can save your work off of your computer so it can't get lost
  • I can give you code help outside of class
  • You can keep the app on your GitHub profile as part of your portfolio of past work

Get Code (Fork)

Here's how to fork

  1. Sign in to GitHub
  2. Go to GitHub atroutt/AndroidBeginnerFirstApp
  3. On the top right press "Fork" to fork the repo to your personal GutHub account
  4. From your fork of the AndroidBeginnerFirstApp repository, click "Clone or download"
  5. Choose 'clone with HTTPS' and copy the url. It should look something like https://github.com/YOUR-USERNAME/AndroidBeginnerFirstApp.git

Get Code (Fork)

  1. On your computer, open a terminal (Mac/Linux) or Git Bash (Windows)
  2. Type git clone, and then paste the URL you copied in Step 5. It will look like this, with your GitHub username instead of YOUR-USERNAME: 
    git clone https://github.com/YOUR-USERNAME/AndroidBeginnerFirstApp.git
  3. Press Enter. Your local clone will be created.
  4. Go to GitHub atroutt/AndroidBeginnerChat and repeat these instructions from Step 3 (fork and clone locally)

Optional: Prep your Android device

If you have an an Android phone enable developer options so that you can install and run your own apps.

An Android phone is not required for this course.

Your phone needs to be running Android Jellybean (4.1-4.3), KitKat (4.4), Lollipop (5.x), or Marshmallow (6.x).

An Android phone is not required for this course, but it is fun to have so you can show your friends what you've built. If you don't have a device you can use the Android emulator with Android Studio. Luckily for you the Android emulators have gotten faster and faster in recent years.

Building your first app

In this section you will learn how to import a new project into Android Studio, and build and run it on your device or emulator.

Download App Code

If you haven't already,
download the source code for the first app (zip).

Alternatively, clone the repo using git:

git clone git@github.com:atroutt/AndroidBeginnerFirstApp.git

PRO TIP: If you fork my repo first you can push your work up to github and keep it in your portfolio. This will also make it possible for me to do code reviews for you outside of class.

Import Project

Launch Android studio

Choose to open an existing Android Studio Project

Choose the AndroidBeginnersFirstApp folder on your computer.

Gradle Sync

Before you can run your first app, you need to make sure you have all the dependencies downloaded. To do that select "Sync Project with Gradle Files" from the top menu bar. This may take a minute.

Run your App (!)

In the top menu choose "Run 'app'"

Android Studio is going to ask you to "Select Deployment Target" — this means choose the device you would like to run on

PRO TIP: Check the box that says "Use same device for future launches."

Project: First Run

Head over to Android Studio and run your first app.
