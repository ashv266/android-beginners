Is this class for you?

You should be curious and excited to learn about how apps are built. You should be prepared to face challenges and have some fun along the way. You are not expected to have any previous Android or even Java experience for this class. However, to move things along it would help if this wasn't your very first introduction to programming in general.

You should have:

  • taken an intro programming class like 'Intro to JavaScript' through Girl Develop It, or
  • be familiar with programming concepts like variables, methods, classes, arrays, conditionals, and loops.
  • and ideally have a basic knowledge of source control with git.

Android Studio

This is the editor we will use to write our apps.

Prep your device

If you have an an Android phone with Android Jellybean (4.1-4.3), KitKat (4.4), Lollipop (5.x), or Marshmallow (6.x), then you need to enable developer options on your device to be able to connect it with your computer to run your apps.

An Android phone is not required for this course, but it is really fun to have. If you don't have a device you can use the Android emulator with Android Studio.
